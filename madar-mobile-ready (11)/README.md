# مشروع مدار — النسخة المختصرة

هذه النسخة تحافظ على واجهة الكود الأول، وتم حذف جميع الأقسام الموجودة أسفل الواجهة الرئيسية مع الإبقاء على التذييل (Footer).

## التشغيل
1. افتحي المجلد في Visual Studio Code.
2. افتحي `index.html`.
3. شغّليه بواسطة Live Server أو Live Preview.


## تعديل المراحل
تم استخدام الجزء المعتمد من الصورة المرجعية للحفاظ على الشكل نفسه، مع مناطق تفاعلية فوق البطاقات وإضاءة عند المرور.


## آخر تعديل
- تم تعديل الواجهة العليا فقط.
- صورة البنت في اليسار.
- بطاقة «كل معادلة...» في اليمين.
- زرا تسجيل الدخول وابدئي التعلم باللون الأخضر.
- أزرار وقسم ابتدائي/متوسط/ثانوي محفوظة دون تغيير.


## آخر تعديل
- نقل شعار مدار إلى اليسار وزر تسجيل الدخول إلى اليمين.
- تصغير بطاقة عبارة «كل معادلة خطوة نحو مستقبلك» وتصغير الخط.
- تحريك عنوان «مدار / منصة الرياضيات التفاعلية» قليلًا إلى اليمين.
- لم يتم تغيير بطاقات ابتدائي ومتوسط وثانوي.
