# مدار (Madar) — منصة الرياضيات التفاعلية

## نظرة عامة
موقع تعليمي ثابت (Static HTML/CSS/JS) بدون خادم backend، باللغة العربية (RTL)، لعرض منصة تعليم رياضيات تفاعلية. يتضمن صفحة رئيسية، تسجيل دخول، إنشاء حساب، وصفحة مراحل تعليمية.

## البنية
- `index.html` / `styles.css` / `script.js` — الصفحة الرئيسية
- `login.html` / `login.css` / `login.js` — تسجيل الدخول
- `register.html` / `register.css` / `register.js` — إنشاء حساب
- `stage.html` / `stage.css` / `stage.js` — صفحة المراحل
- `analytics.html`, `neon-clock.html` — صفحات إضافية
- صور ومقاطع (png/jpg/webp) للعرض المرئي

كل صفحة مستقلة (لا يوجد نظام بناء / bundler)، والملفات تُقرأ مباشرة كملفات ثابتة.

## التشغيل على Replit
- `server.js`: خادم Node.js صغير (بدون أي حزم خارجية) يقدّم الملفات الثابتة من جذر المشروع على المنفذ 5000.
- Workflow: `Start application` يشغّل `node server.js`.
- لا توجد قاعدة بيانات أو أسرار (secrets) مطلوبة حالياً — النماذج (تسجيل الدخول/الحساب) هي واجهات فقط بدون منطق خادم فعلي.

## ملاحظات
- المشروع مستورد من GitHub، تم فقط إضافة خادم ملفات ثابتة (`server.js`) لتشغيله على Replit دون أي تغييرات على الكود أو الهيكلة الأصلية.
